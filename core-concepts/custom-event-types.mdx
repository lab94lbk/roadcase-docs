title: Custom event types
description: Create your own event types with custom names, colors, and icons.
---

Beyond the built-in event types (show, rehearsal, meeting, interview, writing session, hold, and block date), Roadcase lets you create custom event types tailored to your operation.

## Common use cases

Custom event types are useful for events that do not fit neatly into the built-in categories:

- Photo shoots
- Video shoots
- Listening parties
- Album release events
- Fan meet-and-greets
- Studio sessions
- Charity events
- Workshops or clinics

## Creating a custom event type

1. Navigate to **Settings > Event types**
2. Click **New event type**
3. Configure the event type:

| Field | Description |
|---|---|
| **Name** | The display name for this event type (e.g., "Photo shoot") |
| **Color** | A color used to distinguish this event type on the calendar |
| **Icon** | An icon that appears alongside the event type name |

4. Click **Save**

Your custom event type is now available when creating new events.

## Per-artist scoping

Custom event types are scoped to individual artists. If you manage multiple artists, each one has its own set of custom event types. This lets you tailor the available event types to each act's needs.

## Editing and deleting custom event types

You can edit or delete custom event types from **Settings > Event types**. 

<Warning>
  Deleting a custom event type does not delete events that were created with that type. Those events will remain on the calendar but will lose their custom type association.
</Warning>

## Custom event type colors on the calendar

Custom event type colors help you visually distinguish different kinds of events on your calendar. Combined with artist avatar colors (for multi-artist calendars), this gives you a quick visual scan of your schedule.
