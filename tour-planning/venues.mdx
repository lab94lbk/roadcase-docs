Roadcase maintains a venue database so you can quickly associate shows with the right location. Each venue stores essential information that automatically populates when you add it to a show.

## Creating a venue

You can create a venue in two ways:

1. **From a show** — When creating or editing a show, search for a venue. If it does not exist, click **Create new venue** to add it inline.
2. **From venue settings** — Navigate to your venue list and click **New venue**.

### Venue fields

| Field | Description |
|---|---|
| **Name** | The venue name (e.g., "The Fillmore") |
| **Address** | Street address |
| **City** | City |
| **State** | State or province |
| **Country** | Country |
| **Zip** | Postal code |
| **Capacity** | Maximum venue capacity |
| **Phone** | Venue phone number |
| **Website** | Venue website URL |

## Venue search

When creating a show or adding a venue, Roadcase searches real places worldwide — venues, clubs, arenas, and addresses. Start typing a venue name or address and select from the search results to auto-fill location details including:

- **Address fields** — Street, city, state, country, and postal code
- **Geocoding** — Latitude and longitude for mapping
- **Photos** — Browse venue photos and select one as the venue image

## Venue contacts

Each venue can have multiple contacts — the people you communicate with when booking and advancing a show. For each contact, store:

| Field | Description |
|---|---|
| **Name** | Contact person's name |
| **Role** | Their role at the venue (e.g., talent buyer, production manager, box office) |
| **Email** | Contact email |
| **Phone** | Contact phone number |

Venue contacts are available when sharing show advances, making it easy to send advance details to the right person.

## Venue photos

You can set a venue photo from search results. The photo appears on the show detail page when that venue is assigned, giving your team a visual reference for where they are headed.

## Using venues in shows

When you assign a venue to a show, the venue's information is linked — not copied. This means:

- If you update a venue's address, the change reflects on all shows at that venue
- Venue contacts are accessible from any show at that venue
- The venue photo appears on every associated show

## Reusing venues

Once a venue is in your database, it is available for any future show. As you tour more, your venue database grows into a useful reference of every room you have played.
