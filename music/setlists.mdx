Setlists in Roadcase let you plan what you play at every show. Build setlists from your [song library](/music/songs), save templates for reuse, and assign specific setlists to individual shows.

## Building a setlist

To create a setlist:

1. Navigate to **Setlists** in the sidebar
2. Click **New setlist**
3. Give it a name (e.g., "Spring Tour Set" or "Festival 45-min set")
4. Add songs from your library

### Adding songs

Search your song library and add songs to the setlist. Each song entry includes:

| Field | Description |
|---|---|
| **Song** | The song from your library |
| **Order** | Position in the setlist (drag to reorder) |
| **Key** | Defaults to the song's library key — override for this setlist if needed |
| **Tempo** | Defaults to the song's library BPM — override if needed |
| **Duration** | Defaults to the song's library duration — override if needed |

### Per-setlist overrides

You can override the key, tempo, and duration for any song on a specific setlist without changing the values in the song library. This is useful when:

- You play a song in a different key for a particular tour
- The tempo changes for a specific arrangement
- You play an extended version at certain shows

### Reordering songs

Drag and drop songs to change the setlist order. Think about flow — energy levels, key changes, and crowd engagement — when arranging your set.

## Setlist templates

Templates are setlists that are not tied to a specific show. They serve as reusable starting points.

### Creating a template

Any setlist that is not assigned to a show acts as a template. Create it from the setlists page as usual.

### Applying a template to a show

To use a template:

1. Open a show from the calendar
2. Navigate to the setlist section
3. Click **Apply template**
4. Select the template

The template's songs are copied to the show's setlist. You can then customize the show-specific setlist without affecting the template.

## Assigning setlists to shows

Each show can have its own setlist. Assign one from the show detail page. This creates a record of exactly what was played (or planned to be played) at each show.

## Exporting setlists

Click **Export PDF** to generate a printable setlist. The PDF includes:

- Setlist name
- Song list with order numbers
- Key and tempo for each song
- Duration for each song
- Total estimated set length

The PDF is designed for on-stage reference or sharing with the band.

## Who can manage setlists

Both **administrators** and **members** can create, edit, and delete setlists. Crew members can view setlists but cannot modify them. See [Roles & permissions](/core-concepts/roles-and-permissions) for details.
