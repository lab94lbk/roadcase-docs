Todos in Roadcase let you track tasks across your team. Assign work to specific people, set due dates, and scope todos to events or projects.

## Creating a todo

To create a todo:

1. Navigate to **Todos** in the sidebar (or from within a show or project)
2. Click **New todo**
3. Fill in the details:

| Field | Description |
|---|---|
| **Title** | What needs to be done |
| **Assignee** | The team member responsible |
| **Due date** | When it needs to be completed |
| **Event** | Optionally scope this todo to a specific event |
| **Project** | Optionally scope this todo to a [project](/productivity/projects) |

4. Click **Save**

## Assigning todos

Every todo can be assigned to a team member. When you assign a todo, the assignee receives a notification (if they have notifications enabled). This makes it easy to delegate work across your team.

Unassigned todos are visible to everyone but are not specifically anyone's responsibility.

## Due dates and overdue detection

Set a due date to keep tasks on track. Roadcase automatically flags todos that are past their due date as **overdue**. Overdue todos are highlighted in the todo list and surfaced on the [dashboard](/account/dashboard).

## Event-scoped todos

Todos can be scoped to a specific event. This is useful for show-related tasks:

- "Send advance to venue" (scoped to a specific show)
- "Confirm backline rental" (scoped to a show)
- "Book hotel" (scoped to an event)

When viewing an event's detail page, you can see all todos scoped to that event.

## Completing todos

Click the checkbox next to a todo to mark it complete. Completed todos are hidden from the default view but can be shown using filters.

Administrators can mark any todo complete. Members and crew can only complete todos assigned to them.

## Viewing todos

The todo list shows all active (incomplete) todos across your artist. Filter by:

- **Assignee** — See only one person's tasks
- **Due date** — See tasks due soon or overdue
- **Event** — See tasks for a specific event
- **Project** — See tasks within a project
- **Status** — Show or hide completed todos

## Dashboard integration

The [dashboard](/account/dashboard) includes a todo summary widget showing:

- Total active todos
- Overdue count
- Your assigned todos

This ensures tasks do not fall through the cracks.
