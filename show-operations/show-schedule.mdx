Every show in Roadcase can have a detailed schedule that maps out the entire day. The schedule is a timeline of **show items** — individual entries like soundcheck, doors, opener set, headliner set, and anything else that happens during the event.

## Show items

A show item represents a single block on the show schedule. Each item has:

| Field | Description |
|---|---|
| **Name** | What the item is (e.g., "Soundcheck", "Doors", "Headliner set") |
| **Time** | When it starts |
| **End time** | When it ends (optional) |
| **Artist** | Which billing artist this item belongs to (for multi-artist shows) |
| **Order** | Position in the schedule timeline |

## Building a schedule

To build a show schedule:

1. Open a show from the calendar
2. Navigate to the **Schedule** tab
3. Click **Add item** to create a new schedule entry
4. Set the name, time, and optionally assign it to a billing artist
5. Repeat for each block of the day

### Common schedule items

A typical show schedule might look like:

| Time | Item |
|---|---|
| 3:00 PM | Load-in |
| 4:00 PM | Soundcheck |
| 6:30 PM | Doors |
| 7:00 PM | Opener |
| 8:00 PM | Direct support |
| 9:00 PM | Headliner |
| 10:30 PM | Curfew |

## Reordering items

Drag and drop schedule items to reorder them. The timeline automatically updates to reflect the new order.

## Multi-artist shows and billings

When a show has multiple billing artists, you can assign schedule items to specific artists. This makes it clear which artist is responsible for each block:

- Soundcheck times per artist
- Set times per artist
- Load-in/load-out times per artist

Billings define the performance order and are set on the show's main detail page. See [Shows](/tour-planning/shows) for more on billings.

## Schedule notifications

When an administrator updates the show schedule, team members with notifications enabled receive an alert about the change. This ensures everyone stays informed about timing updates. See [Notifications](/account/notifications) for notification preferences.
