If you already have events listed on Bandsintown, Roadcase can import them directly into your calendar. This saves you from manually re-entering show dates, venue details, and locations that already exist on another platform.

## Connecting to Bandsintown

To import events, you need a Bandsintown API key. Roadcase uses this key to access your artist's public event listings through the Bandsintown API.

<Steps>
  <Step title="Navigate to settings">
    Go to **Settings > Integrations** for the artist you want to connect.
  </Step>
  <Step title="Enter your API key">
    Paste your Bandsintown API key into the field provided.
  </Step>
  <Step title="Validate the connection">
    Click **Validate**. Roadcase will look up your artist on Bandsintown to confirm the key works and the correct artist is matched.
  </Step>
</Steps>

## Artist validation

Roadcase automatically matches your artist by name and Bandsintown URL. If the automatic match does not find the right artist — for example, if multiple artists share a similar name — you can provide your Bandsintown artist ID directly. This ensures the correct artist profile is linked before any events are imported.

<Note>
  You can find your Bandsintown artist ID in the URL of your Bandsintown artist page or in your Bandsintown for Artists dashboard.
</Note>

## Previewing events before import

Before importing anything, Roadcase shows you a preview of all upcoming events found on Bandsintown. The preview displays:

| Column | Description |
|---|---|
| **Date** | The event date |
| **Title** | The event or venue name |
| **Venue** | Venue name and city |
| **Status** | Whether the event can be imported or already exists in Roadcase |

Events that match an existing show in Roadcase — by venue name and date, or by a previous import — are flagged as already existing and skipped automatically. You can review the full list before choosing which events to import.

## Importing events

Select the events you want to import and click **Import**. Roadcase creates a show for each selected event, along with the associated venue if it does not already exist. Venue details including name, city, state, country, and coordinates are carried over from Bandsintown.

Imported events become full Roadcase shows, so you can add [schedules](/show-operations/show-schedule), [deals](/show-operations/deals), [advances](/show-operations/advances), [guest lists](/show-operations/guest-lists), and everything else. Events that were previously imported or that match existing shows are skipped to prevent duplicates.

## Continuous sync

After your initial import, Roadcase can automatically sync new events from Bandsintown on an ongoing basis. When continuous sync is enabled, Roadcase periodically checks for new upcoming events on your Bandsintown profile and imports any that do not already exist in your calendar.

This means you can continue managing your public-facing event listings on Bandsintown while Roadcase stays up to date without manual intervention.

<Tip>
  Connect Bandsintown early when setting up your artist in Roadcase. Importing your existing shows saves time compared to entering them manually.
</Tip>

## Venue matching

When importing events, Roadcase attempts to match Bandsintown venues with venues already in your Roadcase account. If a match is found, the imported show is automatically linked to your existing venue record — preserving any contacts, notes, or history you have already built up. If no match is found, Roadcase creates a new venue from the Bandsintown data.

<Note>
  Bandsintown import is one-way — events flow from Bandsintown into Roadcase. Changes you make to an event in Roadcase after import (such as updating the time or adding deal details) are not pushed back to Bandsintown.
</Note>
